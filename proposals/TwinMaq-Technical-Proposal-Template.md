## 3. Scope of Work

### Module 1 — 3D Assets Development

#### 1A. Exterior Assets
- Full 3D exterior model of the building(s) and masterplan
- Integration into an interactive aerial masterplan map
- Photorealistic rendering with accurate lighting, landscaping, and materials
- Day and Night lighting modes

#### 1B. Interior Assets
- 2D interactive floor plans with zoomable, draggable overhead views
- 360° panorama images for each interior unit type
- Photorealistic interior rendering: flooring, wall finishes, furniture
- Unit types to be covered: [List unit types, e.g., 3BHK, 6BHK]
- Consistent furniture and asset library across all unit types

#### 1C. Surrounding Area
- Real-time surroundings map up to **4–5 km radius**
- Key landmarks, roads, schools, hospitals, malls, and mosques labeled
- Interactive pins with travel time and distance data

---

### Module 2 — Home Landing Page

A branded, responsive landing page as the entry point to the TwinMaq platform:

- Aligned with [Client Company]'s brand guidelines and color palette
- Clean, professional layout introducing the project
- Quick-access navigation to all platform sections
- Mobile and tablet optimized
- Supports Arabic and English language toggle

---

### Module 3 — Region & Project Navigation

For clients with multiple projects, a regional landing page is included:

**A. Region Selection Map**
- Saudi Arabia map view with all client projects pinned
- Filter by: Location / Sector / Project Status
- Click on project pin → opens that project's TwinMaq platform

**B. Project Navigation System**
- Intuitive structure for browsing all projects
- Search and filter by type, city, or availability
- Scalable: new projects can be added without redesign

---

### Module 4 — Masterplan & Surroundings

**A. Interactive 2D Location Map**
- Branded map with project pin aligned to brand identity
- Clickable landmarks showing name, category, and distance
- Interactive popup on project pin: name, type, brief description, link

**B. Immersive 3D Surroundings Map**
- Bird's-eye 3D model of project area — 5 km radius
- Realistic 3D environment: roads, buildings, landmarks
- Interactive landmark selection: shows distance and travel time
- Smooth pan, zoom, and rotation controls

**C. 360° Project Bird View**
- Full overhead orbital view of the project
- Zoom in/out (2×) to explore specific zones
- Animated elements: trees, water features, cars, people
- Day / Night mode toggle
- Fixed camera positions per zone (configurable)

---

### Module 5 — Amenities Hotspots

Focused on the unique selling points (USPs) of the project:

- Interactive hotspot pins at key amenity locations (pools, gyms, clubhouses, plazas, lobbies)
- Each hotspot links to a 360° fixed camera view (human scale or aerial)
- Day / Night mood toggle per hotspot
- Numeric navigation: buyers travel between hotspots by clicking numbered pins
- Guided tour flow prevents disorientation
- **Amenities to be covered:** [List from client, e.g., Gym, Pool, Lobby, Cinema, Kids Room]

---

### Module 6 — Unit Selection & Exploration

**A. Interactive Masterplan**
- Color-coded masterplan by unit type (villas, townhouses, apartments, commercial)
- Click on any unit or zone → opens unit detail panel
- Unit info bar: unit code, type, floor, area (m²), availability status

**B. Unit 3D View**
- Clicking a unit opens a 360° orbital exterior view of the unit
- External camera options for tower units (multiple angles)

**C. Floor Selector (for Tower Projects)**
- Select floor → view all units on that floor
- Each unit clickable for detail view

---

### Module 7 — Unit Floor Plan & Layout Exploration

**A. 2D Interactive Floor Plan**
- Top-view zoomable and draggable overhead floor plan
- Floor selector: Ground / First / Roof / All
- Room labels above each space: Living Room, Kitchen, Master Bedroom, etc.
- Area dimensions displayed

**B. 3D Layout Exploration**
- Complete 3D isolation of selected unit
- 360° orbital view of the 3D layout model
- Cross-sectional floor view (cut-through visualization)
- Block layout mode: shows unit boundaries within the building

---

### Module 8 — Materials & Finishes Selector *(TwinMaq Exclusive)*

This feature is not available from any competitor in the Saudi market:

- Buyers select finish categories inside their chosen unit:
  - Flooring (marble, wood, porcelain, etc.)
  - Wall cladding and paint colors
  - Kitchen surfaces and cabinet finishes
  - Bathroom tiles and fixtures
- Real-time 3D preview of selected finishes inside the unit rendering
- Multiple palette options per category (defined by client)
- Save & share configuration link for the buyer
- Export selected finish summary as PDF

**Sales Value:** Creates emotional ownership — buyers invest psychologically in "their" unit.

---

### Module 9 — Amenities 360° Walkthrough

- Interactive exploration of key communal areas
- Major amenities included: [Gym, Pool, Podium, Cinema, Kids Room, Lobby, Lounge — to be confirmed]
- Photorealistic quality: accurate lighting, textures, and finished materials

### Module 10 — Interior Walkthrough Video

**10.1 Interior Walkthrough Video (per unit type)**
- Pre-rendered, cinematic single-camera movement through the unit interior
- Showcases layout, finishes, light, and materials
- Duration: 60–90 seconds per unit type
- Embedded directly in the TwinMaq platform
- Exportable as standalone video for social media / marketing use

### Module 11 — Unit Walkthrough Journey

A guided or freeform virtual tour inside the selected unit:

- Starting point: front door / unit entry
- Pre-set 360° clickable navigation points in each room
- Optional: freeform walk mode (double-tap to navigate)
- Covers: entrance, living room, kitchen, bedrooms, bathrooms, balcony
- iPad and VR headset compatible
- Day / Night lighting toggle inside the unit

---

### Module 12 — User Engagement & Lead Capture

Built-in lead generation tool:

- Buyers can **register interest at the unit level** from within the platform
- Registration form: Name, Phone, Email, Preferred Unit
- Submissions delivered directly to client's sales team via email/CRM
- Admin dashboard shows: total registrations, unit interest breakdown, visitor analytics

## 4. Platform Technical Architecture

```
TwinMaq Platform
├── Web Application (Browser-based — no installation required)
│   ├── Full Desktop Experience (Chrome, Safari, Edge, Firefox)
│   ├── Tablet & iPad Optimized Version
│   ├── Mobile Lite Version
│   └── Touchscreen Kiosk Mode (for sales office)
├── Cloud Infrastructure
│   ├── Secure hosting — 99.9% uptime SLA
│   ├── Daily automated backups
│   ├── CDN (Content Delivery Network) for fast global loading
│   └── SSL encrypted (https) access
├── Admin Dashboard
│   ├── Unit availability status management (real-time)
│   ├── Content update portal
│   ├── Lead registration inbox
│   └── Platform analytics: visitor count, unit views, hotspot engagement
└── Optional VR Module
    ├── Compatible: Meta Quest, HTC Vive
    └── Sales office kiosk configuration available
```

**Supported Devices:** Desktop PC, Mac, iPad, Android tablet, mobile
**Browser Compatibility:** Chrome, Safari, Firefox, Edge (latest versions)
**Language:** Arabic (default) + English (toggle)
