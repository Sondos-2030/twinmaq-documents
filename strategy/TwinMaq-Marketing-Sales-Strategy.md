- Saudi Arabia's real estate sector is growing rapidly under Vision 2030
- Developers are under pressure to pre-sell units before construction completion
- Buyers expect digital-first experiences — static renders and PDFs are losing effectiveness
- No dominant local 3D visualization platform exists — the market is fragmented and underserved
- PropTech adoption is accelerating — timing is ideal for TwinMaq

## 5. Sales Process & Pipeline

### The TwinMaq Sales Funnel

```
Stage 1: LEAD          → Source from database, LinkedIn, referrals, events
Stage 2: QUALIFY       → Confirm active project, budget, decision-maker access
Stage 3: DEMO          → 30-min live walkthrough using Demo.mp4 + live platform
Stage 4: PROPOSAL      → Send Technical Proposal + Financial Proposal (separate docs)
Stage 5: FOLLOW-UP     → 3-touch follow-up sequence over 21 days
Stage 6: NEGOTIATION   → Handle objections, adjust scope if needed
Stage 7: CLOSE         → Signed agreement + 50% payment received
Stage 8: DELIVERY      → 8-week project execution
Stage 9: UPSELL        → Marketing content pack, additional unit types, annual renewal
```

## 7. List-Based Marketing & Outreach Strategy

This is the engine of TwinMaq's growth. All outreach is based on structured, segmented contact lists with defined sequences.

### 7.1 List Structure

Every lead list must be organized with these columns:

| Column | Values |
|--------|--------|
| Company Name | Text |
| Contact Name | Text |
| Title | CEO / Marketing Director / Sales Director / Other |
| Phone (WhatsApp) | Number with country code |
| Email | Text |
| City | Riyadh / Jeddah / Dammam / Khobar / Other |
| Project Name | Known project if any |
| Segment | A / B / C / D |
| Lead Score | 🔴 Hot / 🟠 Warm / 🟡 Cool / ⚪ Cold |
| Source | Database / LinkedIn / Event / Referral |
| Status | New / Contacted / Responded / Demo / Proposal / Closed / Archived |
| Last Contact Date | Date |
| Next Action | Text |
| Notes | Text |

### 7.2 Lead Scoring Criteria

| Score | Label | Criteria |
|-------|-------|----------|
| 🔴 | Hot | Active project launching in <6 months + has marketing team + confirmed budget |
| 🟠 | Warm | Project in development, no launch date yet, open to discussion |
| 🟡 | Cool | Planning stage, 1+ year away, or investor-only company |
| ⚪ | Cold | No active project, no marketing function, or wrong segment |

**Priority:** Always work 🔴 Hot leads first, then 🟠 Warm.

### 7.3 List Segmentation from Existing Database

Files to process:
- `Data/Companies name 2024.xlsx`
- `Data/Copy of Companies name 2024 SS.xlsx`
- `Data/MAYADA DATA - Copy.xlsx`

**Segmentation steps:**
1. Open each file, review all company names
2. Tag each as: Real Estate Developer / Marketing Agency / Investor / Contractor / Other
3. Keep only: Real Estate Developer + Marketing Agency
4. Search LinkedIn for current projects and decision-maker names
5. Add WhatsApp number if available
6. Score each lead (🔴/🟠/🟡/⚪)
7. Export cleaned list as `TwinMaq-Outreach-List-Cleaned.xlsx`

### 7.4 Outreach Batches

Run outreach in weekly batches of 20 leads:

| Batch | Size | Lead Type | Channel |
|-------|------|-----------|---------|
| Batch 1 | 20 | 🔴 Hot | WhatsApp + Email simultaneously |
| Batch 2 | 20 | 🟠 Warm | Email first, WhatsApp follow-up Day 3 |
| Batch 3 | 20 | 🟡 Cool | Email only, once per month |
| Ongoing | 10/week | New leads | Add to appropriate batch |

### 7.5 Multi-Channel Outreach Sequence

Each lead goes through this 5-touch sequence:

```
Day 1:  WhatsApp Message 1 (Arabic intro + Demo.mp4)
Day 1:  Email 1 (English intro + platform overview)
Day 3:  WhatsApp Message 2 (follow-up if no reply)
Day 7:  Email 2 (case study — Aleen project)
Day 10: WhatsApp Message 3 (demo offer)
Day 14: Email 3 (final offer + 14-day pricing validity)
Day 21: Final WhatsApp (closing message)
→ Archive if still no response; re-engage in 90 days
```
